Security posture

Know your threats

CRITICAL
Supply Chain Attacks
Malicious packages injected into open-source dependencies. Mitigation: lockfiles, SBOM generation, and signed releases.
CRITICAL
Container Escape
Exploiting misconfigured containers to breach the host kernel. Mitigation: non-root images, seccomp profiles, Falco runtime rules.
HIGH
Secrets Exposure
API keys hardcoded in repos or leaked in logs. Mitigation: Vault injection, sealed secrets, and automated secret rotation.
HIGH
Privilege Escalation
Over-permissioned IAM roles and RBAC policies. Mitigation: least-privilege by default and OPA policy enforcement.
MEDIUM
Lateral Movement
Compromised pods pivoting across cluster namespaces. Mitigation: NetworkPolicies, service mesh mTLS, zero-trust segmentation.
MEDIUM
Log Injection
Attacker-controlled input corrupting SIEM alerts. Mitigation: log sanitization, structured logging standards, anomaly detection.
